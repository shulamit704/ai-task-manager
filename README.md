# Task Master AI
מחולל תתי-משימות חכם מבוסס בינה מלאכותית (Google Gemini).

## סקירה כללית
פרויקט זה הוא כלי פרודוקטיביות שנועד לעזור למשתמשים לפרק משימות גדולות ומורכבות ל-5 שלבים פשוטים לביצוע. המערכת משתמשת ב-API של Gemini 1.5 Flash כדי לנתח את המשימה ולהפיק תוכנית עבודה מיידית.

## תכונות עיקריות
- פירוק משימות חכם: שימוש ב-LLM מתקדם לניתוח והגדרת שלבי עבודה.
- ממשק משתמש מודרני: התאמה מלאה לעברית (RTL) באמצעות CSS מותאם אישית.
- העתקה מהירה: אפשרות להעתקת השלבים בלחיצת כפתור אחת.
- תמיכה ברשתות מסוננות: הגדרות SSL ייעודיות לעבודה חלקה ברשת נטפרי.

## טכנולוגיות
- Python (Streamlit)
- Google Gemini API
- CSS3 & HTML5

## הוראות הרצה
1. התקנת חבילות: pip install -r requirements.txt
2. הגדרת מפתח API בקובץ ה-Secrets.
3. הרצה: streamlit run app.py
